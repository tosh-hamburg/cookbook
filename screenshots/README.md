# Screenshots Directory

This directory contains screenshots for the main README.md file.

- `web-recipe-list.png` - Recipe overview with filters and search

## Screenshot Guidelines

- **Format**: PNG (preferred) or JPG
- **Resolution**: At least 1280px width for web, 1080px width for mobile
- **Aspect Ratio**: Keep native aspect ratio (16:9 for desktop, 9:16 for mobile)
- **Content**: Show actual recipes and realistic data, avoid test/placeholder content
- **Privacy**: Remove or obscure any personal information

## How to Update

1. Take screenshots following the guidelines above
2. Save them with the exact filenames listed above
3. Place them in this directory
4. The main README.md will automatically reference them
